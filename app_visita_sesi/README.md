# App Visita Técnica SESI — MVP

Aplicativo simples em Streamlit para registrar visitas técnicas de infraestrutura educacional.

## Como rodar no computador

1. Instale o Python.
2. Abra a pasta do projeto no terminal.
3. Instale as dependências:

```bash
pip install -r requirements.txt
```

4. Rode o app:

```bash
streamlit run app.py
```

## Como usar

- Use o menu lateral para acessar qualquer seção em qualquer ordem.
- Clique em **Salvar agora** após preencher uma seção.
- Em **Exportar / Carregar**, gere o relatório HTML.
- Os dados ficam na pasta `dados/`.

## Publicação online

Pode ser publicado no Streamlit Community Cloud usando estes arquivos em um repositório GitHub.
